# 🕋 Hajj Maps — Panduan Perjalanan Suci Versi Bahasa Indonesia

Hajj Maps adalah aplikasi web interaktif yang menampilkan peta perjalanan ibadah haji menggunakan teknologi peta digital. Website ini membantu pengguna memahami lokasi-lokasi penting dalam rangkaian ibadah haji melalui peta, marker lokasi, dan simulasi langkah demi langkah.

## 🌍 Demo

https://dilbar181.github.io/hajj-maps/  

## ✨ Fitur Utama

🗺 **Peta Interaktif**

* Menggunakan Leaflet.js dan OpenStreetMap
* Peta dapat di-zoom dan digeser seperti Google Maps

📍 **Lokasi Penting Haji**
Menampilkan marker lokasi penting seperti:

* Masjid al-Haram (Ka'bah)
* Safa
* Marwah
* Mina
* Arafah
* Muzdalifah
* Jamarat Bridge

🧭 **Sidebar Navigasi**

* Daftar lokasi suci
* Klik lokasi untuk langsung menuju peta

🎬 **Simulasi Haji**

* Mode simulasi langkah demi langkah
* Menampilkan urutan ritual haji
* Marker lokasi aktif dan informasi detail

🛤 **Rute Perjalanan Haji**
Visualisasi rute perjalanan:

Masjid al-Haram → Mina → Arafah → Muzdalifah → Mina → Masjid al-Haram


## 📚 Tujuan Project

Project ini dibuat sebagai media edukasi untuk membantu memahami perjalanan ibadah haji melalui visualisasi peta dan simulasi interaktif.
